### Quick Start

>**_NOTE:_** Você vai precisar dos vetores de testes na pasta ```vetores```, localizada no diretorio ```src```
```console
python3 main.py
```

## Se você quiser adicionar mais casos de teste:
1. Adicione os alogoritmo (normal e de comparacao) em AlgoritmosPerformance.py
2. Os algoritmos de Comparação / Trocas devem retornar os valores nessa mesmas ordem
3. Não é necessário um algoritmo de tempo separado. Implemente só o algoritmo normal
4. Coloque sua função nas listas ALGORITMOS e ALGORITMOS_COUNT
5. Rode: 
```console
python3 main.py
```
6. Os resultados estarão em ```resultados.csv``` separados por pipes
